# Instalando o Zsh

Para instalar o Zsh usando a linha de comando e definir o Zsh como shell padrão é:

## Mac OS

Instalação:
``brew install zsh``


## Debian (Ubuntu, Linux Mint,etc)

Instalação:
``sudo apt install zsh``


## OpenSuse

Instalação:
``sudo zypper install zsh``


## Arch Linux e Manjaro

Instalação:
``sudo pacman -S zsh``


## Fedora

Instalação:
``sudo dnf install zsh``
